方法
======
__construct()
-------------
构造函数

.. code-block:: php

  public function __construct (
      array $config = []
  )


:参数:
  +-------+-------+
  |名称   |说明   |
  +=======+=======+
  |config |配置   |
  +-------+-------+
  
  


get()
-----
获取一个缓存

.. code-block:: php

  public function get (
      string $key,
      mixed $default = null
  ) : mixed


:参数:
  +--------+----------+
  |名称    |说明      |
  +========+==========+
  |key     |键名      |
  +--------+----------+
  |default |默认值    |
  +--------+----------+
  
  


set()
-----
设置一个缓存

.. code-block:: php

  public function set (
      string $key,
      mixed $value,
      \DateInterval|int|null $ttl = null
  ) : bool


:参数:
  +-------+-------------------------------+
  |名称   |说明                           |
  +=======+===============================+
  |key    |键名                           |
  +-------+-------------------------------+
  |value  |值                             |
  +-------+-------------------------------+
  |ttl    |以秒为单位的过期时长           |
  +-------+-------------------------------+
